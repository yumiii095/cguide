❓ 常見問題 (FAQ)

☰

問題一💠要怎麼幫寶可夢回血？

1. 打開 主選單（快捷鍵 C 或 F）

2. 點擊選單優下角的「治療」按鈕

※ 此功能可一次性治療整個隊伍的所有寶可夢，不需逐一處理。

☰

問題二💠我想抓特定的寶可夢，要怎麼知道它會出現在哪裡？

方法1️⃣：利用遊戲內道具

  • 寶可導航：拿在手上點擊右鍵，可查看當前生態域會生成的寶可夢。
  • 寶可蒐尋器：輸入寶可夢英文名稱，可查詢其具體的生成條件。

方法2️⃣：使用指令查詢

輸入指令 /wiki [寶可夢英文名]，系統會顯示該寶可夢的生成生態域、天氣要求與時段。

☰

問題三💠一些常見物品的獲取方式

• 薄荷：在二區或三區的地面上可以採集到種子與葉子進行種植。

• 釣竿模板：通常出現在釣魚時的寶藏，或是在海上的小船木桶內發現。

• 釣竿合成：需使用「模板 + 普通釣竿 + 精靈球」在鍛造台完成合成。

• 幽靈之淚：透過擊殺地獄的幽靈，或是擊殺鬼斯、鬼斯通、耿鬼系列來獲得。

☰

問題四💠二區畫面最上方的「傳奇寶可夢生成剩餘時間」是什麼意思？

當倒數結束時，系統會進行以下動作：

  • 隨機選擇 1 位身處「指定生態域」的玩家生成。
  • 在該玩家腳下生成 1 隻傳奇寶可夢（神獸 / 幻獸 等）。

生成寶可夢特性：

  • 個體值：必定隨機 3V。
  • 其餘數值：全部隨機分配。
☰

問題五💠除了刷神機制，還有其他方式可以獲得傳說寶可夢嗎？

方法一：精靈方塊

  • 敲掉精靈方塊有機率獲得傳說寶可夢
  • 累計 64 次必定出神獸 （/stattrak查看方塊已開啟次數）

精靈方塊獲取方式：

  • 完成 冰果任務（輸入指令 /ob open）
  • 線上獎勵
☰

問題六💠要怎麼把寶可夢給別人？

方法一：交換

  • 面對玩家按 R
  • 左上角點擊「交換」按鈕

方法二：轉換成照片 (此方式目前失效)

1. 使用 /spawn 或菜單傳送至主城

2. 打開菜單 ➜ 點擊「精靈資訊」

3. 滑鼠移到指定寶可夢上 ➜ 右鍵點擊

稍等片刻，寶可夢會轉換成 照片道具 出現在背包中

☰

問題七💠我可以去哪裡蓋家？

如何前往：

1. 打開菜單（按 C）

2. 點擊 「傳送菜單」

3. 選擇 「家園世界」

家園世界說明：

  • 唯一不會洗白的世界，不用擔心建築物消失。
  • 可在此設置領地（指令 /res）並建立自己的家。

⚠️ 注意事項：

  • 家園世界 無法使用隨機傳送 (RTP)。
  • 建議準備長距離移動手段（如傳送工具或飛行坐騎）。

🔰 入門教學指南

⭐ 新手開局指南

1. 加入伺服器先往營前方小攤車領取新手福利。

2. 按 M 選擇初始御三家就可以去主世界冒險了。

3. 如果寶可夢等級太低可以先去 EVS 訓練（按 C 呼叫選單內即可找到）。

💡 伺服器五大區域說明

🏰 主城

查看活動、奇蹟交換、挑戰道館、參與抽獎。

🏡 家園世界

不會洗白！建議在此建立領地蓋家。

⚔️ 冒險 / 地獄 / 終界

各分為「二區」與「三區」：

  • 二區：定期刷新傳說寶可夢。畫面上方會顯示神獸刷新地點，倒計時結束前待在指定地點，神獸會生在玩家腳下。(未來神獸刷新改為三區)
  • 三區：可遇到野生神獸，並有突襲活動（目前維修中）。

領取你的第一隻夥伴

進入伺服器後，按下鍵盤上的 M 鍵，即可開啟選單選擇你的御三家寶可夢。選定後按下 R 鍵即可將它放出來與你一同冒險。

免費福利領取

可以到 Discord 攻略區領取免費 5V（僅新手）。

保護你的家園 (領地)

使用「木鋤頭」選取兩個對角點後，輸入 /res create [名稱] 即可建立領地，防止其他玩家破壞。
※ 注意：只能在家園世界建立領地蓋家，因為二區三區會定期洗白。(二區未來會轉家園世界，三區不變)
你可以使用 /res padd [玩家ID] 給予好友權限 或 /res pset [玩家ID]單獨設定該玩家權限。
